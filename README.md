# ritikakumar0204

Personal portfolio site for Ritika Kumar. Sections cover About, Education, Experience, Projects, Research, Leadership, Achievements, and Contact.

## Tech stack

- Vite + React + TypeScript
- Tailwind CSS + shadcn/ui
- Deployed on Vercel

## Local development

```sh
npm install
npm run dev
```

The dev server runs on http://localhost:8080 (configured in `vite.config.ts`).

## Build

```sh
npm run build      # production build to dist/
npm run preview    # preview the built output
```

## Deployment

Push to `main` and Vercel auto-deploys. Build command and output dir are set in `vercel.json`.
